# Student Management System

A React-based Student Management System that allows users to add and delete student records.

## Features

- Add students
- Delete students
- Responsive design
- Empty state handling
- Modern UI with Tailwind CSS

## Tech Stack

- React
- Tailwind CSS
- Vite

## Live Demo

[https://student-management-system-six-inky.vercel.app/]

## Installation

npm install
npm run dev
